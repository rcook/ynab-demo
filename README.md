# ynab-demo by Richard Cook

Demo of decoding YNAB register data using Cassava and Haskell

## Clone repository

```
git clone https://github.com/rcook/ynab-demo.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec ynab-demo
```

## Run tests

```
stack test
```
